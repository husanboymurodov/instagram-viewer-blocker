Privacy Policy

Last updated: May 2026

Overview

Instagram Viewer & Blocker is a browser extension that allows users to view profile information and block Instagram accounts — including accounts that have blocked you. This privacy policy explains what data the extension accesses and how it is used.

Data collection

Instagram Viewer & Blocker does not collect, store, transmit, or share any personal data. Specifically:

  • No data is sent to any external server operated by the extension developer.
  • No analytics, tracking, or telemetry of any kind is performed.
  • No user account information is stored outside the user's own browser.

How it works

The extension operates entirely within the user's browser. When a user requests to block an account or look up profile information, the extension makes requests directly to Instagram's own servers using the user's existing logged-in session — the same requests Instagram's website makes itself. No third party is involved.

Permissions

  • instagram.com host access — required to interact with Instagram's internal API from within the user's browser session.
  • cdninstagram.com / fbcdn.net host access — required to fetch profile pictures from Instagram's CDN so they can be displayed in the extension popup.
  • tabs — required to read the current tab's URL so the username field can be auto-filled when visiting a profile page.
  • cookies / activeTab / scripting — required to inject the content script that performs profile lookups and block operations within the Instagram session context.
